Analysis

Analysis Folders

Project outputs are kept in three folders in the shared ISWEL project folder pdrive\is-wel\indus\analysis

  • key_outputs – Contains key figures chosen from all the outputs.
  • all-outputs – Contains all data output from the ISWEL MESSAGEix Indus model.
  • output_code – Contains code used to process outputs.
